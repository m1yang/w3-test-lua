# timer dialog
```lua
ac.game:timerDialog(title: string[, timer/number])
    -> timer dialog
```

```lua
player:timerDialog(title: string[, timer/number])
    -> timer dialog
```

### setTitle
```lua
td:setTitle(string)
```

### setTimer
```lua
td:setTimer(timer/number/nil)
```

### getTimer
```lua
td:getTimer()
    -> timer
```

### remove
```lua
td:remove()
```
